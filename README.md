# PROYECTO: Sistema de Gestión de Notas

## Descripción:

Este proyecto consiste en un programa desarrollado en C++ que permite gestionar estudiantes y sus respectivas notas. A través de un menú interactivo, el usuario puede registrar, consultar y analizar información académica de manera sencilla.

## Objetivo:

El objetivo del programa es facilitar el registro y manejo de estudiantes junto con sus calificaciones, permitiendo realizar operaciones básicas como búsqueda, cálculo de promedios y visualización de datos.

## Contenido / Funcionalidades:

* Registro de estudiantes con nombre y apellido
* Ingreso y validación de notas (0 a 20)
* Mostrar lista de estudiantes con sus notas
* Cálculo del promedio general
* Búsqueda de estudiantes por nombre completo
* Identificación del estudiante con la mayor nota
* Menú interactivo con múltiples opciones

## Estructura:

El proyecto está organizado de la siguiente manera:

* `proyecto1.cpp`: Contiene todo el código fuente del programa
* Funciones separadas para cada operación (registrar, mostrar, buscar, etc.)
* Uso de arreglos para almacenar nombres y notas
* Uso de funciones para modularizar el programa

## Herramientas utilizadas:

* Lenguaje de programación: C++
* Compilador: g++
* Entorno de desarrollo: Visual Studio Code
* Control de versiones: Git y GitHub
